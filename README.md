# acidcam-cli

Command Line Version of Acid Cam for Linux with OpenCV:

 Command Line Arguments

	-l List filters

	-i input video

	-o output video

	-f filter list

	-v image visible


Example:

	$ acidcam -i input.mov -o output.mov -f 1,2,3

Or to view progress:

	$ acidcam -i input.mov -o output.mov -f 0 -v

To List filters:

	$ acidcam -l
